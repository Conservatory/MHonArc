NEWS

Syntax

Envariable

M2H_NEWS=[ 0 | 1 ]

Element(s)


<NEWS>
<NONEWS>

Command-line Option(s)


-news
-nonews

Description

When NEWS is on, newsgroups listed in the Newsgroup field of a message header will be converted to news URL hyperlinks.

Default Setting

On.

Resource Variables

N/A

Examples

None.

1.0
