MHonArc Resource List

OUTDIR

Syntax

Envariable

M2H_OUTDIR=path

Element

N/A

Command-line Option

-outdir path

Description

OUTDIR defines the location of the archive to process.

Default Setting

.

Aka, the current working directory.

Resource Variables

N/A

Examples

    % mhonarc -add -outdir /home/ehood/archive inbox
