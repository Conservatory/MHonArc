MHonArc Resource List

DEFRCFILE

Syntax

Envariable

M2H_DEFRCFILE=filename

Element

N/A

Command-line Option

N/A

Description

DEFRCFILE is the pathname to the default resource file. The file is read each time MHonArc is invoked. The file is read before processing the resource file specified by the RCFILE resource and before command-line options are processed. The default resource file allows you change the default values resource values of MHonArc.

Default Setting

user-home-path/DEFRCNAME

If the above does not exist, then:

MHonArc-library-path/DEFRCNAME

Resource Variables

N/A

Examples

None.

2.0
