Syntax

Envariable

M2H_SUBSORT=[ 0 | 1 ]

Element(s)

<SUBSORT>
<NOSUBSORT>

Command-line Option(s)

-subsort
-nosubsort

Description

When SUBSORT is active, messages are listed by subject in the main index.

Default Setting

Off.

N/A

None.

1.0
