DOCURL

Syntax

Envariable

M2H_DOCURL=URL

Element

<DOCURL>
 URL
</DOCURL>

Command-line Option

-docurl URL

Description

DOCURL defines the URL to the MHonArc documentation. The value of DOCURL can be accessed by the $DOCURL$ resource variable when defining resource values.

Default Setting

http://www.mhonarc.org/

Resource Variables

N/A

Examples

None.

1.0
